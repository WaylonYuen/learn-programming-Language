# Selection sort 選擇排序法

前置知識：[#for...loop]()、[#swap]()

## Explain

選擇排序是反覆進行「搜尋數列中的最小(大)值，將它與最左邊的數對調」的操作。搜尋數列中的最小(大)值使用[線性搜尋]()。
